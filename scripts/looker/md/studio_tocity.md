#  TOCITY

Returns the city name of the specified Google Ads criteria ID.
## Sample usage
`TOCITY(Criteria ID)`
## Syntax
`TOCITY( X )`
### Parameters
  * - A field that provides a valid Google Ads criteria ID for a city


## Examples
Example formula | Input | Output  
---|---|---  
` TOCITY(City Code) ` |  1023191  |  New York
